# AWS 3-Tier Web Application

## Project Overview

Designed and deployed a highly available 3-tier web application using AWS Cloud services.

### AWS Services Used

- Amazon VPC
- EC2
- Application Load Balancer (ALB)
- Auto Scaling
- Amazon RDS
- Route 53
- CloudFront
- AWS WAF
- CloudWatch
- IAM
- Security Groups
